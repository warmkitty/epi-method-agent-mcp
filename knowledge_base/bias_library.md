# 偏倚类型库

> 版本：v1.0 | 所有研究设计前必须系统检查此表

---

## 偏倚分类体系

```
偏倚
├── 选择偏倚（Selection Bias）
│   ├── 选择偏倚（狭义）
│   ├── 健康工人效应
│   ├── 损耗偏倚（Loss to follow-up）
│   ├── 信息删失偏倚（Informative censoring）
│   └── 入选偏倚（Inclusion bias）
│
├── 信息偏倚（Information Bias）
│   ├── 错分偏倚（Misclassification）
│   │   ├── 非差异错分（Non-differential）
│   │   └── 差异错分（Differential）
│   ├── 回忆偏倚（Recall bias）
│   ├── 观察者偏倚（Observer bias）
│   └── 检测偏倚（Detection bias / Surveillance bias）
│
├── 混杂偏倚（Confounding）
│   ├── 指示性混杂（Confounding by indication）
│   ├── 测混杂（Measured confounding）
│   └── 未测混杂（Unmeasured confounding）
│
├── 时间相关偏倚（Time-related Bias）
│   ├── 不朽时间偏倚（Immortal time bias）
│   ├── 时滞偏倚（Time-lag bias）
│   ├── 就诊间隔偏倚（Visit frequency bias）
│   └── 日历时间偏倚（Calendar-time bias）
│
└── 特殊偏倚（RWE特有）
    ├── 竞爰风险（Competing risk）
    ├── 易感人群耗竭（Depletion of susceptibles）
    ├── 编码漂移（Coding drift / Upcoding）
    ├── 治疗改变偏倚（Treatment switching bias）
    ├── 处方偏好偏倚（Physician preference bias）
    └── 数据截断偏倚（Administrative censoring）
```

## 详细偏倚表

### 一、选择偏倚类

#### 1. Confounding by indication（指示性混杂）
| 字段 | 内容 |
|------|------|
| **发生机制** | 治疗选择由疾病严重程度/预后因素驱动，这些因子同时影响结局 |
| **对结果方向的可能影响** | 取决于适应症：重症患者用药→效应被低估（阴性偏倚）；轻症患者用药→效应被高估（阳性偏倚） |
| **诊断方法** | 检查治疗组与对照组基线特征是否平衡；检验未测混杂变量（吸烟、BMI）是否存在 |
| **应对策略** | IPTW/OW、匹配（PSM）、分层、G-formula；明确残余混杂方向 |
| **残余风险** | 高（未测混杂因子如疾病严重程度评分、吸烟、BMI通常不可测） |

#### 2. Healthy worker effect（健康工人效应）
| 字段 | 内容 |
|------|------|
| **发生机制** | 参保且能接受某干预的人群比一般人群更健康 |
| **对结果方向的可能影响** | 高估药物效果（与真实世界一般人群相比） |
| **诊断方法** | 比较研究人群与全源人群的基线特征分布 |
| **应对策略** | 外部数据链接；明确研究人群代表性局限 |
| **残余风险** | 中（外部数据获取困难） |

#### 3. Informative censoring（信息删失偏倚）
| 字段 | 内容 |
|------|------|
| **发生机制** | 删失不是随机的，而是与潜在的结局风险相关 |
| **对结果方向的可能影响** | 通常低估风险（高死亡风险患者提前退出随访） |
| **诊断方法** | 比较删失组与未删失组的基线特征；Kaplan-Meier删失曲线 |
| **应对策略** | IPW for censoring；Fine-Gray竞争风险模型 |
| **残余风险** | 中（假设可交换性仍可能不成立） |

#### 4. Selection bias（选择偏倚，狭义）
| 字段 | 内容 |
|------|------|
| **发生机制** | 纳入/排除标准导致研究人群不能代表目标人群 |
| **对结果方向的可能影响** | 方向不确定，取决于选择标准 |
| **诊断方法** | 比较纳入人群与目标人群的基线特征 |
| **应对策略** | 放宽纳排标准；外部效度分析；敏感性分析（不同纳排标准） |
| **残余风险** | 中 |

---

### 二、信息偏倚类

#### 5. Misclassification（错分偏倚）
| 字段 | 内容 |
|------|------|
| **发生机制** | 诊断编码错误、药物识别错误、结局定义错误 |
| **对结果方向的可能影响** | 非差异错分→偏向null（效应低估）；差异错分→方向不确定 |
| **诊断方法** | 数据质量评估；与金标准（EMR/病历）比对；敏感性分析（不同定义阈值） |
| **应对策略** | 提高定义严格度（以平衡灵敏度/特异度）；多定义敏感性分析 |
| **残余风险** | 中高（尤其是特异性低的定义） |

#### 6. Detection bias / Surveillance bias（检测偏倚）
| 字段 | 内容 |
|------|------|
| **发生机制** | 暴露组比对照组接受更多随访/检查，导致结局更可能被发现 |
| **对结果方向的可能影响** | 高估风险（阳性偏倚） |
| **诊断方法** | 比较两组随访频率、检查次数 |
| **应对策略** | 阴性对照结局（negative control outcome）；标准化随访 |
| **残余风险** | 中 |

---

### 三、时间相关偏倚类

#### 7. Immortal time bias（不朽时间偏倚）⚠️ 最重要
| 字段 | 内容 |
|------|------|
| **发生机制** | 从队列进入到首次暴露之间的"不朽时间"被错误地算入暴露组；或time zero定义错误 |
| **对结果方向的可能影响** | 几乎总是高估治疗获益（阳性偏倚），幅度可达HR=0.5以上 |
| **诊断方法** | 检查time zero定义；检查暴露组在index前是否有空白期；绘制时间线 |
| **应对策略** | 正确定义time zero（用药开始日=time zero）；或使用landmark分析；new-user design |
| **残余风险** | 低（正确设计可基本消除） |

#### 8. Calendar-time bias（日历时间偏倚）
| 字段 | 内容 |
|------|------|
| **发生机制** | 治疗组和对照组在不同时间段入组，期间诊疗标准/背景治疗发生变化 |
| **对结果方向的可能影响** | 方向不确定 |
| **诊断方法** | 比较两组入组时间分布 |
| **应对策略** | 在模型中控制日历时间（年/月作为协变量）；分层；DiD设计 |
| **残余风险** | 低（充分控制后） |

#### 9. Visit frequency bias（就诊频率偏倚）
| 字段 | 内容 |
|------|------|
| **发生机制** | 高频就诊患者更可能被诊断、更可能检测到结局 |
| **对结果方向的可能影响** | 高估风险（如癌症检出率） |
| **诊断方法** | 比较两组年均就诊次数 |
| **应对策略** | 标准化随访密度；阴性对照 |
| **残余风险** | 中 |

---

### 四、竞争风险与特殊偏倚

#### 10. Competing risk（竞争风险）
| 字段 | 内容 |
|------|------|
| **发生机制** | 一个类型的事件（如死亡）阻止了另一个事件（如心肌梗死）的发生或观察 |
| **对结果方向的可能影响** | 传统KM/Cox会高估累积发生率 |
| **诊断方法** | 检查竞争事件发生率；绘制CIF曲线 |
| **应对策略** | Fine-Gray模型；Aalen-Johansen估计量 |
| **残余风险** | 低（正确方法可处理） |

#### 11. Depletion of susceptibles（易感人群耗竭）
| 字段 | 内容 |
|------|------|
| **发生机制** | 高风险患者在随访早期发生事件，剩余人群风险降低，导致随时间推移效应减弱（或反向） |
| **对结果方向的可能影响** | 时间依赖的效应异质性，可能掩盖真实效应 |
| **诊断方法** | 检查事件发生率随时间的变化；Schoenfeld残差检验 |
| **应对策略** | 时间分层分析；时变系数模型 |
| **残余风险** | 中 |

#### 12. Coding drift / Upcoding（编码漂移/高编码）
| 字段 | 内容 |
|------|------|
| **发生机制** | 诊断/手术编码实践随时间变化（如DRG支付改革后） |
| **对结果方向的可能影响** | 产生虚假的时间趋势，干扰组间比较 |
| **诊断方法** | 检查编码率随时间的变化（尤其政策节点前后） |
| **应对策略** | 时间分层；排除政策节点前后数据；敏感性分析 |
| **残余风险** | 中高 |

#### 13. Treatment switching（治疗切换偏倚）
| 字段 | 内容 |
|------|------|
| **发生机制** | 初始分配的治疗在随访中发生改变（交叉） |
| **对结果方向的可能影响** | 稀释治疗效应（意向治疗分析时） |
| **诊断方法** | 检查交叉率；绘制治疗持续曲线 |
| **应对策略** | ITT + per-protocol双重分析；逆概率删失加权（IPCW） |
| **残余风险** | 中（交叉原因可能与预后相关） |

## 偏倚诊断检查清单

每个研究设计完成后，必须逐项检查：

```
□ Confounding by indication — 治疗组/对照组基线是否可比？
□ Immortal time bias — time zero是否正确定义？
□ Informative censoring — 删失是否与潜在结局相关？
□ Misclassification — 主要变量的编码质量如何？
□ Competing risk — 是否有竞争事件？
□ Detection bias — 两组随访/检测频率是否平衡？
□ Calendar-time bias — 两组入组时间分布是否平衡？
□ Depletion of susceptibles — 事件率是否随时间异常下降？
□ Coding drift — 编码实践在研究期间是否发生变化？
□ Unmeasured confounding — 有哪些重要变量无法测量？
```

## 残余偏倚的定性/半定量说明

在proposal中，残余偏倚应用以下框架描述：

```
残余偏倚评估格式：

偏倚类型 | 方向（可能） | 幅度估计 | 对结论的影响 | 是否可以定量评估
---------|------------|---------|------------|----------------
指示性混杂（未测） | 阳性偏倚（高估获益） | 中等（E-value ≈ 2.0） | 结论可能不成立 | 否，需E-value敏感性分析
错分偏倚（诊断） | 偏向null | 小（特异度>90%） | 结论保守，仍成立 | 是，多定义敏感性分析
不朽时间偏倚 | 已消除（正确设计） | - | - | -
```
