# 医保/EMR/注册数据能力边界

> 版本：v1.0 | 适用：中国真实世界数据场景

---

## 一、数据类型总览

| 数据类型 | 典型来源 | 核心优势 | 核心局限 |
|---------|---------|---------|---------|
| **医保结算数据** | 国家/省级医保局 | 覆盖广、费用完整、纵向随访 | 缺乏临床指标、诊断编码粗糙 |
| **电子病历（EMR）** | 医院HIS系统 | 临床指标丰富、医嘱详细 | 覆盖局限、随访受医院约束 |
| **肿瘤注册数据** | 肿瘤登记中心 | 发病率权威、分期可信 | 更新滞后、治疗信息不全 |
| **死亡登记数据** | CDC/卫健委 | 死因权威 | 编码质量参差不齐 |
| **患者登记研究** | 主动设计收集 | 变量完整、随访主动 | 样本小、偏倚风险（参与偏倚） |
| **多源链接数据** | 医保+EMR+死亡 | 互补优势 | 链接率、链接误差 |

## 二、医保数据能力矩阵

### 2.1 可以直接捕捉

| 维度 | 可捕捉变量 | 说明 |
|------|----------|------|
| **人群识别** | 年龄、性别、参保类型、地区 | 人口学变量完整 |
| **疾病诊断** | ICD-10诊断编码（主诊+副诊） | 每次就诊的诊断列表 |
| **治疗暴露** | 药品编码、处方日期、用药天数、剂量 | 适用于门诊/住院处方 |
| **医疗资源利用** | 住院次数、住院天数、门诊次数、急诊次数 | |
| **医疗费用** | 总费用、药品费、检查费、手术费 | 按结算类别分 |
| **手术/操作** | 手术编码（ICD-9-CM-3） | |
| **随访时间** | 持续参保期间的所有就诊记录 | |

### 2.2 间接推断（质量受限）

| 维度 | 推断方式 | 局限 |
|------|---------|------|
| **疾病严重程度** | 住院频率、特定检查费用、联合用药 | 粗糙，与临床评分相关性弱 |
| **治疗依从性** | MPR/PDC计算 | 无法确认实际服药 |
| **疾病进展** | 二线药物切换、特定并发症诊断 | 编码延迟、遗漏 |
| **死亡** | 结算停止推断 | 误判率较高，建议链接死亡登记 |

### 2.3 通常无法捕捉

| 缺失维度 | 影响 | 应对策略 |
|---------|------|---------|
| **实验室指标** | 无HbA1c/LDL/eGFR等 | 链接EMR；或用代理变量（相关药物使用） |
| **临床评分** | 无NYHA分级、ECOG评分等 | 链接EMR；或专家判断局限 |
| **遗传/基因信息** | 无 | 不可替代 |
| **症状和体征** | 无结构化记录 | 无替代 |
| **患者报告结局** | 无 | 需专门PRO数据 |
| **肿瘤分期** | 无或不可靠 | 链接肿瘤注册 |
| **吸烟/饮酒** | 无 | 主要残余混杂来源 |
| **BMI/体重** | 无 | 主要残余混杂来源 |
| **社会经济状况** | 有限（参保类型代理） | 参保类型作为粗代理 |

## 三、EMR数据能力矩阵

### 3.1 相比医保的优势

| 优势维度 | 具体内容 |
|---------|---------|
| **实验室检验** | 血常规、生化、肿瘤标志物、特殊检测 |
| **影像/病理** | 结构化报告（质量参差） |
| **临床评分** | 部分医院有结构化评分录入 |
| **医嘱详情** | 给药剂量、频次、途径更精确 |
| **入院/出院诊断** | 比医保诊断更精确 |

### 3.2 EMR的核心局限

| 局限 | 影响 |
|------|------|
| **随访受限于就诊** | 转诊、院外死亡无法捕捉 |
| **单中心/多中心偏倚** | 代表性问题（三甲医院偏选） |
| **数据标准化差异** | 不同医院字段名、编码不统一 |
| **结构化率低** | 大量信息在非结构化文本中（需NLP） |
| **历史数据完整性** | 早期EMR录入不完整 |

## 四、中国特殊情境说明

### 4.1 医保数据分层

```
全国层面：国家医保局数据（研究可及性低）
    ↓
省级层面：省级医保局（如北京、上海、广东）可申请研究
    ↓
市级层面：地市级医保局（覆盖相对完整的地区人群）
```

### 4.2 北京医保数据特殊说明

- **覆盖率**：覆盖北京参保人群（约2000万+），含职工和居民
- **费用完整性**：门诊/住院/药店三类结算记录
- **诊断编码**：ICD-10，但编码行为受医院/科室影响较大
- **药品编码**：使用国家医保药品编码（不同于医院内部编码）
- **时间范围**：通常可回溯5-10年（视申请协议而定）
- **主要局限**：无实验室数据、无临床评分、异地就医可能丢失

### 4.3 常见诊断识别策略

```
策略1：主诊断 ≥ 1 次（灵敏度高，特异度低）
策略2：主诊断 ≥ 2 次（平衡）
策略3：主诊断 + 相关药物（特异度高）
策略4：住院主诊断（最高特异度）
策略5：算法验证（与EMR/注册数据比对）
```

## 五、数据适用性快速判断框架

```
研究问题 → 所需关键变量 → 该变量是否可从目标数据获取？

可获取 → 信效度如何？ → 高/中 → 可直接使用
                     → 低    → 需敏感性分析/外部验证

不可获取 → 是否有代理变量？ → 有 → 评估代理效度，标注局限
                           → 无 → 该分析目标不可行，降级设计
```
