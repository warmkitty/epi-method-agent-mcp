## 一、方法选择决策树

```
研究目标是什么？
│
├── 描述性分析
│   ├── 患病率/发病率 → 标化率、Poisson/负二项回归
│   ├── 治疗模式（LOT） → LOT算法、Sankey图、序列分析
│   ├── 依从性 → MPR/PDC、Kaplan-Meier停药曲线
│   └── 医疗资源利用 → 计数模型（Poisson/负二项）、成本GLM
│
├── 比较有效性（单时点暴露）
│   ├── 结局为二元 → Logistic回归、风险差RD
│   ├── 结局为时间-事件（随访完整） → Cox PH模型
│   ├── 需控制混杂 → PS匹配、IPTW、OW、G-formula
│   └── 高维混杂 → TMLE、SuperLearner
│
├── 比较安全性（不良事件）
│   ├── 自对照设计 → SCCS（Self-Controlled Case Series）
│   ├── 队列设计 → Cox（考虑竞争风险用Fine-Gray）
│   └── 罕见事件 → 精确 logistic、Firth惩罚回归
│
├── 经济负担/成本分析
│   ├── 总成本 → GLM（Gamma族+log连接）/ 二部模型
│   ├── 增量成本 → 匹配+回归调整
│   └── 长期预测 → Markov模型、分区生存模型
│
├── 预算影响分析（BIA）
│   └── 人群规模 × 使用率 × 单人成本 × 情景差异
│
└── 预测模型
    ├── 风险评分 → Cox、机器学习（需外部验证）
    └── 分类 → Logistic、随机森林、XGBoost（需验证）
```

## 二、常用方法详细说明

### 2.1 倾向评分方法

| 方法 | 估计目标 | 优点 | 缺点 | 何时使用 |
|------|---------|------|------|---------|
| **1:1 PS匹配** | ATT | 直观、易解释 | 损失样本、匹配后仍需加权 | 样本量大、对照组充足 |
| **IPTW** | ATE | 保留全样本、易于实现 | 极端权重问题 | 标准方法 |
| **OW（重叠权重）** | ATO | 无极端权重、更稳定 | 稍难解释 | **推荐默认**，尤其样本分布重叠不足时 |
| **PS分层** | ATE（近似） | 简单 | 信息损失 | 敏感性分析 |
| **熵平衡** | ATE | 精确平衡矩条件 | 实现复杂 | 高维协变量平衡 |

**权重修剪（Trimming）建议**：
- IPTW：修剪超出[1/10, 10]的权重
- OW：无需修剪（天然限制在0-1之间）

---

### 2.2 高级因果推断方法

#### G-formula（G-computation）
- **适用**：时变混杂（confounding by time-varying confounders affected by prior treatment）
- **原理**：标准化直接计算，预测每个干预下的结局
- **实现**：回归预测 + 边际化
- **优点**：直观、可处理复杂干预
- **缺点**：模型依赖性强（需正确指定结果模型）

#### TMLE（Targeted Maximum Likelihood Estimation）
- **适用**：高维数据、双重稳健估计需求
- **原理**：初始估计 + 靶向更新（ targeting step）
- **优点**：双重稳健、统计效率最高、可处理机器学习预测器
- **缺点**：实现复杂、需要超参数调优、结果解释门槛高
- **何时使用**：高维协变量（>50）、IPTW/OW结果不稳定时

#### Difference-in-Differences（DiD）
- **适用**：政策干预评估、准自然实验
- **假设**：平行趋势假设（parallel trend）
- **实现**：回归：`Y = β0 + β1·Treat + β2·Post + β3·Treat×Post + ε`
- **中国医保场景**：DRG支付改革前后、药品谈判前后

#### IV（工具变量）
- **适用**：存在未测混杂，且有合适工具变量
- **工具变量条件**：与暴露相关、与结局独立（除通过暴露）、不影响混杂因子
- **RWE中常用IV**：医师处方偏好、地区处方率、医保政策变化时点
- **局限性**：IV估计LATE（局部平均效应），解释需谨慎

### 2.3 生存分析方法

| 方法 | 适用场景 | 关键假设 |
|------|---------|---------|
| **Kaplan-Meier** | 描述性生存曲线 | 独立删失 |
| **Cox PH** | 主要比较方法 | 比例风险假设（PH假设） |
| **Fine-Gray** | 存在竞争风险 | 竞争风险独立性 |
| **Aalen-Johansen** | 竞争风险累积发生率（非参数） | — |
| **Landmark分析** | 处理时变暴露 | 需预设landmark时间点 |
| **IPTW-Cox** | 加权Cox模型 | 权重在risk set中正确应用 |

**PH假设检验方法**：
- 图形法：log(-log(S(t)))曲线
- 检验法：Schoenfeld残差检验
- 若PH不成立：分层Cox、时变系数Cox、分段时间分析

### 2.4 成本/资源利用分析方法

#### 成本数据特点
- 严重右偏（大量零值+长右尾）
- 不符合正态分布

#### 推荐方法

**方法一：GLM（Gamma族 + log连接）**
```text
适用：无零值的成本数据
模型：E[Y|X] = exp(Xβ)  （Gamma分布）
优点：直接估计平均成本；不需要变换后反变换
```

**方法二：二部模型（Two-part model）**
```text
Part 1：是否有任何成本（Logistic回归）
Part 2：在有成本的人群中，成本大小（GLM-Gamma）
最终：E[Y] = P(Y>0) × E[Y|Y>0]
适用：大量零值（如门诊成本）
```

**方法三：对数变换 + 反变换（smearing estimator）**
```text
不推荐（偏差较大，除非样本量极大）
```

### 2.5 敏感性分析标准套餐

| 分析类型 | 必须包含的敏感性分析 |
|---------|-------------------|
| 比较有效性（IPTW/OW） | 1. 不同PS模型设定；2. 不同权重修剪阈值；3. 未测混杂E-value |
| 生存分析 | 1. 不同随访截断；2. PH假设不满足时的替代模型；3. 竞争风险分析 |
| 成本分析 | 1. 不同GLM分布族；2. 不同零值处理；3. 极端成本截断（99%分位） |
| 诊断定义 | 1. 宽定义 vs 严定义；2. 主诊断 vs 主次诊断；3. 住院 vs 门诊+住院 |
| 偏倚方向 | 定性讨论残余偏倚对效应估计的可能方向 |

## 三、方法选择禁忌

```
❌ 不要在观察性研究中报告OR并解释为"治疗效应"
   → 正确：报告HR/RD，并明确说明识别假设

❌ 不要在没有检查PH假设的情况下使用Cox模型
   → 正确：先检验PH，不满足时分段时间或分层Cox

❌ 不要对右偏成本数据使用普通OLS线性回归
   → 正确：使用GLM Gamma或二部模型

❌ 不要对二分类结局使用线性概率模型（LPM）而不检查边界问题
   → 正确：使用logistic + 边际效应，或直接报告RD

❌ 不要直接比较KM曲线而不考虑竞争风险（当竞争事件存在时）
   → 正确：使用Fine-Gray或Aalen-Johansen

❌ 不要在没有new-user设计的情况下比较活跃用药者
   → 正确：使用new-user design或明确标注prevalent user bias

❌ 不要盲目使用TMLE（高维数据才需要）
   → 正确：先尝试IPTW/OW，只在必要时升级
```
