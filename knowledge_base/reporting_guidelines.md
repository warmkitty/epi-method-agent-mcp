## 一、STROBE 声明（观察性研究报告规范）

**适用**：队列研究、病例对照研究、横断面研究

**22个条目核心检查清单**：

| 章节 | 条目 | 关键要求 |
|------|------|---------|
| **标题和摘要** | 1 | 在标题或摘要中识别研究设计 |
| **引言** | 2 | 解释科学背景和原理 |
| **方法** | 3 | 研究设计（回顾性/前瞻性、匹配、权重） |
|  | 4 | 研究人群（纳排标准、数据来源） |
|  | 5 | 暴露/干预变量的定义和测量方法 |
|  | 6 | 结局变量的定义和测量方法 |
|  | 7 | 混杂因素的定义和测量方法 |
|  | 8 | 样本量（如适用） |
|  | 9 | 定量变量处理（分类/连续） |
|  | 10 | 统计方法（包括混杂控制、敏感性分析） |
| **结果** | 11 | 研究人群 Flowchart（STROBE Figure 1） |
|  | 12 | 描述性数据（基线特征表） |
|  | 13 | 结局数据（事件数、人时、发生率） |
|  | 14 | 主要结果（未调整 + 调整 + 加权结果） |
|  | 15 | 辅助分析（敏感性分析、亚组） |
| **讨论** | 16 | 关键结果小结 |
|  | 17 | 研究结果的局限性（偏倚、混杂、通用性） |
|  | 18 | 结果与其他研究的可比性 |
|  | 19 | 研究意义（对临床/政策的含义） |
| **其他信息** | 20-22 | 资助、伦理、数据可得性 |

**RWE论文检查重点**：
- 必须报告 PS 平衡性检验（SMD表）
- 必须报告权重分布（IPTW的权重范围）
- 必须包含偏倚讨论（至少3种主要偏倚）
- 必须报告敏感性分析结果

## 二、RECORD 声明（常规数据研究报告规范）

**适用**：使用 routinely-collected health data（医保、EMR、注册数据）的观察性研究

**在 STROBE 基础上的额外要求**：

| 条目 | 要求 |
|------|------|
| **数据来源** | 明确说明数据来源、覆盖人群、时间范围、数据获取协议 |
| **数据链接** | 如有多源链接，说明链接方法、链接率、链接质量评估 |
| **数据质量** | 编码实践、完整性、历史变化 |
| **变量定义** | 提供详细的算法定义（附录或GitHub） |
| **缺失数据** | 报告缺失模式、处理方式 |
| **可重复性** | 提供分析代码（如可能） |

**中国RWE研究特别建议**：
- 明确说明医保数据覆盖地区、年份、参保类型
- 提供诊断/药品编码列表（附录）
- 说明数据使用是否经过伦理审查

## 三、ISPOR 任务组报告规范

### 3.1 ISPOR Good Research Practices for CER（比较有效性研究）

**方法学质量标准**：

```
GRACE 评分体系（≥4分/5分项目为高质量）：
1. 研究问题明确性
2. 研究设计适当性
3. 人群定义适当性
4. 比较组选择适当性
5. 结局定义适当性
6. 混杂控制充分性
7. 分析适当性
8. 结果解释适当性
9.  funding透明度
10. 研究注册/预注册
```

### 3.2 ISPOR-SRWG 预算影响分析规范

**BIA报告必须包含**：
- 目标人群规模（及来源）
- 使用率/覆盖率（及趋势）
- 单人成本（分解：药品费、管理费、监测费）
- 时间范围（通常5-10年）
- 折现率（如适用）
- 情景分析（乐观/基准/悲观）
- 敏感性分析（关键参数±20%）

## 四、FDA RWE 相关框架

### 4.1 FDA 21st Century Cures Act / RWE Framework

**RWE可用于**：
- 已批准药物的新适应症
- 已批准药物的新人群
- 上市后安全性监测（主动监测）

**RWE递交要求（2023+）**：
- 研究方案须预注册（protocol registration）
- 主要分析须在数据提取前确定（pre-specified）
- 数据来源须经过验证（data source validation）
- 偏倚控制须有明确的敏感性分析

### 4.2 NMPA RWD/RWE 指导原则（中国）

**核心要求**（2020年版，2023年更新）：
- RWD须经适用性评估（fit-for-purpose）
- 研究方案须明确 estimand
- 主要分析人群须明确（ITT/PP/per-protocol）
- 敏感性分析须覆盖主要偏倚来源
- 结论须有明确的不确定性表述

## 五、ROBINS-I 偏倚风险评估

**用于非随机干预研究的偏倚风险评估工具**

| 偏倚领域 | 评估内容 |
|---------|---------|
| **混杂偏倚** | 是否有充分的混杂控制？ |
| **受试者选择偏倚** | 受试者的分配/参与是否引入偏倚？ |
| **干预分类偏倚** | 干预暴露是否准确测量？ |
| **偏离干预偏倚** | 是否有交叉/停药，如何处理？ |
| **数据缺失偏倚** | 缺失数据是否随机？ |
| **结局测量偏倚** | 结局测量是否受干预状态影响？ |
| **报告偏倚** | 是否有选择性报告？ |

**评估结论格式**：
```
低风险 / 中风险 / 高风险 / 严重风险 / 无信息

偏倚总评：若任何领域为"高风险"或"严重风险"，
          则研究结论的因果解释力显著受限。
```

## 六、E-value 敏感性分析（未测混杂量化）

**定义**：使得观察效应值（如HR=0.8）变为不显著（CI包含1）所需的最小未测混杂因子强度

**报告格式**：
```
观察HR = 0.82 (95% CI: 0.74-0.91)
E-value（点估计）= 2.25  即：需要未测混杂与暴露的RR≥2.25、
                             且与结局的RR≥2.25，
                             才能解释观察效应。
E-value（置信下限）= 1.87
```

**解释**：
- E-value大 → 结果对未测混杂稳健
- E-value接近1 → 结果很容易被未测混杂解释

## 七、Proposal/Protocol 必备章节结构

```
1. Background（研究背景）
2. Research Question（研究问题 + PICO/PECO）
3. Study Rationale（研究理由 + 知识缺口）
4. Study Design（研究设计图 + TTE框架）
5. Data Source（数据来源 + 适用性评估）
6. Study Population（纳排标准 + 流程图）
7. Exposure / Intervention Definition（暴露定义）
8. Outcome Definition（结局定义）
9. Covariates（混杂变量列表）
10. Statistical Analysis（统计方案）
    10.1 描述性分析
    10.2 主要分析
    10.3 混杂控制方法
    10.4 敏感性分析
    10.5 亚组分析
11. Bias Assessment（偏倚评估表）
12. Sample Size / Power（样本量）
13. Ethical Considerations（伦理）
14. Timeline（时间计划）
15. References（参考文献）
```
